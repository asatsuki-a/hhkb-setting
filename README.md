# hhkb-setting
AutoHotKeyみたいな設定を入れられないときに**JIS配列HHKB**だけでWindowsをMacっぽく操作するための設定


## こだわりポイント
- Fnキーとセットでemacsキーバインドっぽいカーソルの操作
- Aキーの左(本来はCtrl)をFnキーに設定
- Ctrlキーは親指で押すのでMacモードのCmdキーの位置に設定
